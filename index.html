Main Page

The programming language is a set of commands, written according to rules that are determined by the programming language, and then these commands go through several stages until they are executed on a computer. Programming can be defined as the process of writing instructions and commands for a computer or any other device, to direct it and inform it of how to deal with data Or how to carry out a series of required actions.

Based on its proximity to human languages, programming languages are divided into:
  1. A high-level language (close to the language human beings understand) such as: C Sharp, Python and Java.
  2. - Low level language (close to machine language) such as: assembly language.

Sometimes it is divided according to desired purposes from the language used. There are languages that are designed to work on specific devices, such as for a company to produce a computer or central processor (central processing unit), and provide it with a user guide containing the commands that are executed on it, and there are other more general languages that work independently of the type of machine, that is, It works within a virtual machine, such as the Java language.
